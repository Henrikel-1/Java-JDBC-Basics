📌 Sobre o projeto

Este repositório contém exemplos e exercícios práticos utilizando JDBC (Java Database Connectivity) em Java.

O objetivo do projeto é consolidar os fundamentos de acesso a banco de dados relacional sem o uso de frameworks ORM, abordando:

Conexão com banco de dados

Execução de comandos SQL

Uso de PreparedStatement

Manipulação de ResultSet

Implementação de operações CRUD

O projeto possui caráter educacional e foi desenvolvido como parte dos estudos de persistência de dados e integração Java ↔ Banco de Dados.
